# Contexto para la IA: Copiloto de Desarrollo - Impetus Mindset

Actúa como un Senior WordPress & Front-End Developer. Me vas a ayudar a maquetar la nueva web de **Impetus Mindset** usando WordPress + Elementor. Para mantener el sitio rápido, utilizaremos la menor cantidad de plugins posibles y resolveremos los diseños complejos con **Custom CSS** limpio, semántico y mobile-first que yo pegaré en Elementor.

## 👤 El Cliente y el Tono
- **Cliente:** Iván Melchor, NLP Coach e Ingeniero Informático. 25+ años de experiencia.
- **Público Objetivo:** Empresas, líderes de equipos y profesionales que buscan Equipos de Alto Rendimiento.
- **Tono:** Profesional, directo, humanista, tecnológico pero cercano ("No fluff").

---

## 🏗️ Guía de Secciones y Contenido Real para Maquetación

### 1. HERO SECTION
- **Titular Principal:** Liderazgo, comunicación y habilidades humanas.
- **Subtitular:** Transformamos a responsables en líderes, y a grupos de empleados en Equipos de Alto Rendimiento.
- **CTA:** "Te invito a que colaboremos" (Scroll a contacto).

### 2. SOBRE IVÁN & MÉTRICAS
- Texto enfocado en su dualidad (Informático + Coach). 
- **Destacar frase:** "Tomé conciencia de que entiendo mejor a las personas con las que colaboro que a las máquinas que estas utilizan".
- **Contadores (Grid de 3 o 4 columnas):**
  - `10+` Años de experiencia en Coaching/PNL.
  - `150+` Proyectos entregados.
  - `60+` Clientes Satisfechos.
  - `Incontables` Cafés.

### 3. SERVICIOS (Formato Grid o Cards Limpias)
Debemos estructurar los 4 pilares que ofrece:
1. **Formación:** Liderazgo y equipos de alto rendimiento / Comunicación efectiva.
2. **Seguimiento Individual Post-Formación:** Acompañamiento e integración real de nuevos hábitos.
3. **Coaching y Mentoría Estratégicos:** Intervención de alta precisión para perfiles clave usando PNL y Agile/Waterfall.
4. **Packs Exclusivos:** Combinación eficiente (Formación + Coaching + Mentorías).

### 4. SOCIAL PROOF / TESTIMONIOS (Nueva Sección)
- Diseñar una sección limpia (estilo Bento Grid o Testimonials Slider) para colocar las referencias de los clientes.

### 5. SOCIAL MEDIA HUB (Triple Columna - Nueva Sección)
Necesito el HTML/CSS estructural para meter en un contenedor de Elementor dividido en 3 columnas:
- **Columna 1 (Blog):** Una card moderna que simule el último post del blog (Imagen destacada, título, fecha y botón "Leer más").
- **Columna 2 (Instagram):** Placeholder/Contenedor responsivo para un iframe de Instagram.
- **Columna 3 (YouTube):** Placeholder/Contenedor responsivo para un reproductor de video de YouTube.

### 6. PRICING & TARIFAS (Nueva Sección)
- Diseñar una tabla de precios de 3 columnas para los "Packs Exclusivos":
  - Plan 1: Formación + Coaching
  - Plan 2: Formación + Mentorías (Destacado)
  - Plan 3: Pack Premium (Formación + Coaching + Mentorías)

### 7. CONTACTO
- Formulario limpio con campos: Nombre, Email, Mensaje y Botón de envío.

## 🛠️ Primera Tarea Requerida
Ayúdame a generar las **Variables CSS globales** (colores premium para un perfil tech/coaching, fuentes sans-serif limpias y espaciados consistentes) y el código CSS para estructurar la sección **Social Media Hub (Triple Columna)** de forma que sea 100% responsiva. Dame solo el CSS y las clases a usar. colores como un azul cobalto oscuro o gris grafito de fondo (identidad tech/seria) combinado con un azul grisáceo para los acentos y CTAs (identidad de crecimiento/coaching).
