# Impetus Mindset - Rediseño Web 2026

Repositorio para la migración, limpieza y rediseño de la plataforma web de Iván Melchor (**Impetus Mindset**), enfocado en Liderazgo, Comunicación y Coaching/PNL para perfiles tecnológicos y corporativos.

## 🎯 Objetivos del Proyecto
1. **Unificación de Idioma:** Eliminar por completo los textos *dummy* en inglés de la plantilla anterior.
2. **Estructura limpia:** Convertir la landing en un embudo de conversión B2B (Empresas y Profesionales).
3. **Nuevas Secciones Integradas:**
   - Hub de Redes Sociales (3 columnas: Blog, Instagram, YouTube).
   - Testimonios / Referencias de clientes.
   - Tabla de Tarifas y Packs Exclusivos.

## 🛠️ Stack Local
- Laragon + WordPress 6.x
- Tema: Hello Elementor
- Maquetador: Elementor (Estructura base) + Custom CSS vía IA.

## 📋 Módulos de la Landing Page
- **Hero:** Liderazgo, comunicación y habilidades humanas.
- **Sobre Iván:** Trayectoria de Ingeniero a Coach/NLP.
- **Métricas:** Contador dinámico (10+ Años, 150+ Proyectos, 60+ Clientes).
- **Servicios:** Formación, Seguimiento Individual, Coaching y Packs Exclusivos.
- **Social Proof:** Referencias y Comentarios (Nuevo).
- **Social Media Hub:** Blog + IG + YouTube (Nuevo).
- **Pricing:** Tarifas y Planes (Nuevo).
- **Contacto:** Formulario limpio.
